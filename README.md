# 🌍 GeoTraining

Jogo de mapa para treinar geografia: clique em um país e acerte a bandeira e o nome.

**▶️ Jogar:** https://brunotertuliano.github.io/world-map-quiz/

## Recursos

- Todos os países e territórios, com polígonos reais (Natural Earth)
- Português e inglês
- 3 níveis de dificuldade: Fácil, Médio e Difícil

Feito com [D3](https://d3js.org/) + [TopoJSON](https://github.com/topojson). Bandeiras via [flagcdn](https://flagcdn.com/).
